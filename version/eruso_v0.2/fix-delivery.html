🔧 배송 상태 "대기중" 문제 해결

처방전 상태는 "완료"인데 배송 상태가 "대기중"으로 표시되는 문제를 해결합니다.

⚠️ 현재 문제

  • 발행일: 2024. 02. 11. 오후 08:24
  • 배송 상태: 대기중 (잘못됨)
  • 예상 상태: 배송완료 (완료된 처방전)

해결 방법 (자동 수정)

1단계: 문제 확인

현재 데이터베이스의 상태 불일치를 확인합니다.

2단계: 자동 수정

sync.html 페이지에서 "배송 정보 동기화 실행" 버튼을 클릭합니다.

3단계: 결과 확인

환자 포털 또는 약국 포털에서 배송 상태가 정상적으로 표시되는지 확인합니다.

수동 수정 (고급 사용자)

특정 처방전의 배송 상태만 수정하고 싶으시면 아래 도구를 사용하세요.

예방 방법

앞으로 이런 문제가 발생하지 않도록 하려면:

  • 약국 포털에서 "배송 시작" 버튼을 클릭하여 배송을 시작하세요
  • 환자 포털에서 QR 인증으로 수령을 완료하세요
  • 상태가 자동으로 업데이트되지 않으면 sync.html로 수정하세요
