⚡ 빠른 시작 가이드

  1. 배송 상태 문제 해결: auto-fix.html 열기 → 버튼 클릭
  2. 샘플 데이터 추가: test.html 열기 → "샘플 데이터 추가" 클릭
  3. 약국 완료 검증: verify.html 열기 → 자동 검증
  4. 환자 포털 접속: index.html 열기 → 로그인
