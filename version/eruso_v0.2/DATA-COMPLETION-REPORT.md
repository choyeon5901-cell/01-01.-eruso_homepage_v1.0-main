## 🌟 주요 기능

### add-data.html의 핵심 기능

#### 1. 병원 정보 추가
- ✅ 병원명, 주소, 전화번호, 면허번호
- ✅ 담당자 정보 (이름, 이메일, 전화번호)
- ✅ 자동 사용자 계정 생성 (role: hospital)
- ✅ 자동 UUID 생성
- ✅ 등록 일시 자동 기록

#### 2. 약국 정보 추가
- ✅ 약국명, 주소, 전화번호, 면허번호
- ✅ 약사 정보 (이름, 이메일, 전화번호)
- ✅ 자동 사용자 계정 생성 (role: pharmacy)
- ✅ 자동 UUID 생성
- ✅ 등록 일시 자동 기록

#### 3. 환자 정보 추가
- ✅ 이름, 이메일, 전화번호, 주소
- ✅ 생년월일 (선택사항)
- ✅ 자동 사용자 계정 생성 (role: patient)
- ✅ 자동 UUID 생성
- ✅ 등록 일시 자동 기록

#### 4. 데이터 조회 및 관리
- ✅ 실시간 통계 대시보드
  - 등록된 병원 수
  - 등록된 약국 수
  - 등록된 환자 수
  - 병원 담당자 수
- ✅ 병원 목록 조회 (표 형식)
- ✅ 약국 목록 조회 (표 형식)
- ✅ 환자 목록 조회 (표 형식)
- ✅ 데이터 새로고침 기능

## 🎯 사용 시나리오

### 시나리오 1: 신규 병원 등록
```
1. add-data.html 열기
2. "병원 추가" 탭 선택
3. 정보 입력:
   - 병원명: 조치원 중앙병원
   - 주소: 세종특별자치시 조치원읍 군청로 93
   - 전화번호: 044-860-1000
   - 면허번호: H2024-001
   - 담당자명: 김의사
   - 담당자 이메일: doctor@hospital.com
   - 담당자 전화번호: 010-1234-5678
4. "병원 추가" 버튼 클릭
5. ✅ 성공 메시지 확인
6. "데이터 조회" 탭에서 등록 확인
```

### 시나리오 2: 신규 약국 등록
```
1. add-data.html 열기
2. "약국 추가" 탭 선택
3. 정보 입력:
   - 약국명: 조치원 건강약국
   - 주소: 세종특별자치시 조치원읍 문화로 123
   - 전화번호: 044-865-2000
   - 면허번호: P2024-001
   - 약사명: 이약사
   - 약사 이메일: pharmacist@pharmacy.com
   - 약사 전화번호: 010-2345-6789
4. "약국 추가" 버튼 클릭
5. ✅ 성공 메시지 확인
6. "데이터 조회" 탭에서 등록 확인
```

### 시나리오 3: 신규 환자 등록
```
1. add-data.html 열기
2. "환자 추가" 탭 선택
3. 정보 입력:
   - 이름: 박환자
   - 이메일: park@example.com
   - 전화번호: 010-5678-9012
   - 주소: 세종특별자치시 조치원읍 서부로 456, 202동 801호
4. "환자 추가" 버튼 클릭
5. ✅ 성공 메시지 확인
6. "데이터 조회" 탭에서 등록 확인
```


### 2. 주소 입력 규칙
```
형식: [시/도] [구/군] [읍/면/동] [도로명] [번지], [건물명] [동] [호]
예시: 세종특별자치시 조치원읍 서부로 123, 101동 501호
```

### 3. 이메일 중복 방지
- 각 사용자는 고유한 이메일 사용
- 도메인별로 구분 권장
  - 병원: @hospital.com
  - 약국: @pharmacy.com
  - 환자: @example.com

### 4. 전화번호 형식
```
병원/약국 대표 전화: 044-XXX-XXXX
개인 전화: 010-XXXX-XXXX
```
