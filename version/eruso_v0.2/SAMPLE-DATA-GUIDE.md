## 🚀 사용 방법

### 방법 1: CSV 파일 사용 (권장)

1. **sample_hospitals.csv 파일 사용**
   ```
   프로젝트 폴더에 sample_hospitals.csv 파일이 생성되었습니다.
   ```

2. **bulk-upload.html로 일괄 등록**
   ```
   1. bulk-upload.html 열기
   2. sample_hospitals.csv 파일 업로드
   3. 미리보기 확인
   4. "일괄 등록 실행" 클릭
   ```

3. **완료**
   ```
   성공: 3건 (서울아산내과의원 포함)
   실패: 0건
   ```

---

### 방법 2: csv-validator.html로 검증 후 등록

1. **CSV 검증**
   ```
   1. csv-validator.html 열기
   2. sample_hospitals.csv 업로드
   3. "검증 시작" 클릭
   ```

2. **검증 결과**
   ```
   ✅ 정상: 3건
   ❌ 오류: 0건
   ⚠️ 경고: 0건
   ```

3. **일괄 등록**
   ```
   "일괄 등록 계속하기" 버튼 클릭
   → bulk-upload.html로 자동 이동
   → 등록 실행
   ```

---

### 방법 3: add-data.html로 개별 등록

1. **add-data.html 열기**

2. **병원 추가 탭 선택**

3. **정보 입력**
   - 병원명: `서울아산내과의원 내과`
   - 주소: `세종특별자치시 조치원읍 새내로 97 2 7층`
   - 전화번호: `044-862-7550`
   - 면허번호: `H2024-002`
   - 담당자명: `가정의학과`
   - 담당자이메일: `seoulasan1@hospital.com`
   - 담당자전화번호: `010-1111-2223`

4. **"병원 추가" 버튼 클릭**

## 🔄 전체 워크플로우 테스트

### 1단계: 병원 등록 (완료 가능)
```
✅ sample_hospitals.csv 사용
✅ 서울아산내과의원 내과 등록
```

### 2단계: 약국 등록
```
1. bulk-upload.html에서 약국 템플릿 다운로드
2. 약국 정보 입력
3. 업로드 및 등록
```

### 3단계: 환자 등록
```
1. bulk-upload.html에서 환자 템플릿 다운로드
2. 환자 정보 입력 (테스트용 2-3명)
3. 업로드 및 등록
```

### 4단계: 처방전 발행 테스트
```
1. 병원 로그인 (가정의학과)
2. hospital.html에서 처방전 발행
3. 환자 선택
4. 약물 정보 입력
5. 발행
```

### 5단계: 약국 처리 테스트
```
1. 약국 로그인
2. pharmacy.html에서 처방전 확인
3. 접수 → 조제 → 배송 시작
4. QR 코드 자동 생성 확인
```

### 6단계: 환자 수령 테스트
```
1. 환자 로그인
2. patient.html에서 처방전 확인
3. QR 인증으로 수령
4. 완료 상태 확인
```
