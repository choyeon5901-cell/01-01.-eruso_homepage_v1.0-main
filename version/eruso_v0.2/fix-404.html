🚨 API 404 오류

API 응답 오류: 404 Not Found

RESTful Table API에 접근할 수 없습니다.

📋 문제 원인

이 프로젝트는 Genspark의 RESTful Table API를 사용하도록 설계되었습니다.

하지만 현재 환경에서는 API에 접근할 수 없습니다:

  • 로컬 파일 시스템에서 실행 중 (file://)
  • API 서버가 실행되지 않음
  • 네트워크 연결 문제

✅ 해결 방법 1: Genspark 플랫폼에서 배포

권장 방법

이 프로젝트는 Genspark 플랫폼에서 배포되어야 정상 작동합니다.

  1. Genspark의 Publish 탭으로 이동합니다
  2. "Publish" 버튼을 클릭합니다
  3. 배포된 URL에서 프로젝트를 실행합니다
  4. RESTful Table API가 자동으로 연결됩니다

🔧 해결 방법 2: 로컬 개발 환경 (임시)

⚠️ 임시 해결책

로컬에서 테스트하려면 LocalStorage를 사용하는 버전으로 전환할 수 있습니다.

※ 주의: 로컬 모드는 브라우저의 LocalStorage만 사용하며, 실제 API는 사용하지 않습니다.

💻 해결 방법 3: 웹 서버 실행

로컬 웹 서버 사용

file:// 프로토콜 대신 http:// 프로토콜로 실행합니다.

방법 A: Python 사용

# Python 3
python -m http.server 8000

# 또는 Python 2
python -m SimpleHTTPServer 8000

→ 브라우저에서 http://localhost:8000 접속

방법 B: Node.js 사용

npm install -g http-server
http-server -p 8000

방법 C: VS Code Live Server

  1. VS Code 확장 프로그램 "Live Server" 설치
  2. index.html 우클릭 → "Open with Live Server"

⚠️ 중요 안내

  • 이 프로젝트는 Genspark 플랫폼용으로 설계되었습니다
  • 로컬에서는 RESTful Table API가 작동하지 않습니다
  • 완전한 기능을 사용하려면 Publish 탭에서 배포하세요
  • 로컬 테스트는 UI/UX 확인용으로만 사용하세요
