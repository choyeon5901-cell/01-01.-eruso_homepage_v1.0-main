주요 기능

  • CSV 템플릿 다운로드
  • 드래그 앤 드롭 파일 업로드
  • 데이터 미리보기
  • 빠른 일괄 등록
  • 실시간 진행 상황 표시

사용 팁

  1. 템플릿을 다운로드하여 형식을 확인하세요
  2. Excel 또는 텍스트 편집기로 데이터를 입력하세요
  3. UTF-8 인코딩으로 저장하면 한글이 깨지지 않습니다
  4. 파일을 업로드하고 미리보기를 확인하세요
  5. "일괄 등록 실행" 버튼을 클릭하세요
