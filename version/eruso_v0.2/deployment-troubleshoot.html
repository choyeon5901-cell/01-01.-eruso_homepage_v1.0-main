1️⃣ 개발자 도구 열기

2️⃣ Console 탭 확인

🔴 흔한 에러 메시지:

❌ 404 Not Found - tables/users ❌ Failed to fetch ❌ CORS error ❌ Cannot read property 'data' of undefined

3️⃣ Network 탭 확인

🔍 확인해야 할 파일들:

  • css/style.css - 스타일시트
  • js/common.js - JavaScript
  • tables/users - 사용자 데이터
  • tables/prescriptions - 처방전 데이터

일반적인 문제와 해결 방법

❌ 문제 A: "API 응답 오류: 404 Not Found"

증상: 데이터가 로드되지 않음, 빈 목록

원인: RESTful Table API 경로 문제

해결:

  1. Publish 탭에서 다시 배포
  2. 배포된 URL이 Genspark 도메인인지 확인
  3. 외부 호스팅(Vercel, Netlify)에 배포했다면 API가 작동하지 않습니다
API 404 오류 해결 도구

❌ 문제 B: 스타일이 깨짐 (레이아웃 망가짐)

증상: 버튼, 카드, 색상이 이상함

원인: CSS 파일을 찾을 수 없음

해결:

  1. Network 탭에서 css/style.css 요청 확인
  2. 404 에러가 있다면 파일 경로 문제
  3. 모든 HTML 파일에서 경로가 href="css/style.css"인지 확인

❌ 문제 C: 버튼이 작동하지 않음

증상: 클릭해도 반응 없음

원인: JavaScript 로드 실패

해결:

  1. Console 탭에서 JavaScript 오류 확인
  2. Network 탭에서 js/common.js 로드 확인
  3. 브라우저 캐시 삭제 후 재시도 (Ctrl + Shift + Delete)

❌ 문제 D: 데이터가 비어있음

증상: API는 작동하지만 목록이 비어있음

원인: 샘플 데이터가 없음

해결:

  1. test.html 페이지로 이동
  2. "샘플 데이터 추가" 버튼 클릭
  3. 포털 페이지로 돌아가서 새로고침
샘플 데이터 추가하기

긴급 지원

위 방법으로 해결되지 않으면:

  1. 스크린샷 찍기: 개발자 도구의 Console과 Network 탭
  2. URL 확인: 배포된 사이트의 주소를 복사
  3. 에러 메시지 복사: 빨간색 에러 메시지 전체를 복사
  4. 도움 요청: 위 정보와 함께 문제 설명
