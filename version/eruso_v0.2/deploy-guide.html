원인: 이 프로젝트는 정적 웹사이트이지만 Cloudflare Workers로 배포하려고 시도했습니다.

⚠️ 중요 안내

이 프로젝트는 Genspark의 RESTful Table API를 사용하므로:

  • 외부 호스팅 서비스에서는 API가 작동하지 않습니다
  • Genspark 플랫폼 내에서 배포해야 합니다
  • 또는 로컬 모드로 제한적으로 사용 가능합니다

⭐ 권장: Genspark 플랫폼 배포

이것이 유일하게 완전히 작동하는 방법입니다!

1단계: Publish 탭 이동

Genspark 인터페이스에서 "Publish" 탭을 찾습니다.

2단계: 배포 버튼 클릭

"Publish" 또는 "Deploy" 버튼을 클릭합니다.

3단계: URL 받기

배포가 완료되면 고유한 URL을 받습니다.

4단계: 사용 시작

해당 URL에서 프로젝트를 사용합니다.

✅ 장점:
  • RESTful Table API 완전 지원
  • 모든 기능 정상 작동
  • 데이터 영구 저장
  • 어디서나 접근 가능

⚠️ 대안: 로컬 모드 (제한적)

외부 배포가 불가능한 경우 사용합니다.

사용 방법

  1. fix-404.html 열기
  2. "로컬 모드 활성화" 버튼 클릭
  3. 페이지 새로고침
  4. index.html에서 로그인

⚠️ 단점: 브라우저 LocalStorage만 사용, 데이터 공유 불가, 제한적 기능

📚 참고: 다른 정적 호스팅 서비스

⚠️ 주의: 아래 방법들은 RESTful Table API를 지원하지 않습니다!

🔧 문제 해결

Cloudflare Workers 배포 실패

원인: 이 프로젝트는 정적 웹사이트입니다. Cloudflare Workers가 아닌 Pages를 사용해야 합니다.

해결: Cloudflare Pages 또는 다른 정적 호스팅 서비스를 사용하세요.

중요: 그러나 RESTful Table API는 Genspark에서만 작동합니다!

API 404 오류

원인: RESTful Table API에 접근할 수 없습니다.

해결: Genspark에서 배포하거나 로컬 모드를 사용하세요.

fix-404.html 열기
권한 오류

메시지: "Please check your Cloudflare permissions"

해결: 이 오류는 무시하세요. 정적 호스팅 방법을 사용하면 됩니다.
